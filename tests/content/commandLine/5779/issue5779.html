Issue 5779: Allow to set the root element in $ and $$

Steps to reproduce

  1. Select and enable the Console panel
  2. Execute following expressions in the command line and check the results
    'expression' -> 'result'
    1. $('div') -> <div class="test">
    2. $$('div') -> [div.test, div#root, div.rootdiv1, div.rootdiv2]
    3. $('div', document.getElementById('root')); -> <div class="rootdiv1">
    4. $$('div', document.getElementById('root')); -> [div.rootdiv1, div.rootdiv2]
