Issue 5951: Title of array-like objects should be clickable

Steps to reproduce

  1. Select and enable the Console panel
  2. Execute following expression in the command line:
    document.getElementsByTagName("span")
  3. The result should be:
    HTMLCollection[span.test, span#root]
  4. Click the HTMLCollection title. You should be navigated to the DOM panel.
