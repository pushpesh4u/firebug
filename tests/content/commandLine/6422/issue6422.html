Issue 6422: Add a right click option for accessing objects in the Command Line

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. In the Command Line, run: $p
    → The result should be undefined.
  4. In the Command Line, run: 0
  5. Right-click the result and select "Use In Command Line"
    → $p should be entered into the Command Line, fully selected.
  6. Press enter
    → The result should be 0.
  7. In the Command Line, run: console.log("%o", "");
  8. Enter a into the Command Line, without pressing enter
  9. Right-click the logged string and select "Use In Command Line"
    → $p should be appended to the Command Line, and selected.
  10. Remove the a, and press enter
    → The result should be "".
