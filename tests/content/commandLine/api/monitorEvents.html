Command Line API: monitorEvents()

Single event

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Focus the Command Line
  4. Type monitorEvents(document.getElementById("monitoredElement"), "click") and hit Enter
  5. Click the blue <div> (#monitoredElement) above

Expected result

  • ⇒ The console should list a click event:
    click clientX=<x coordinate>, clientY=<y coordinate>

Event family

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Focus the Command Line
  4. Type monitorEvents(document.getElementById("monitoredElement"), "key") and hit Enter
  5. Click the blue <div> (#monitoredElement) above
  6. Press A on the keyboard

Expected result

  • ⇒ The console should list a keydown, a keypress and a keyup event:
    keydown charCode=0, keyCode=65 keypress charCode=97, keyCode=0 keyup charCode=0, keyCode=65

Multiple events

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Focus the Command Line
  4. Type monitorEvents(document.getElementById("monitoredElement"), ["click", "key"]) and hit Enter
  5. Click the blue <div> (#monitoredElement) above
    ⇒ The console should list a click event:
    (click clientX=<x coordinate>, clientY=<y coordinate>)
  6. Press A on the keyboard
    ⇒ The console should list a keydown, a keypress and a keyup event:
    keydown charCode=0, keyCode=65 keypress charCode=97, keyCode=0 keyup charCode=0, keyCode=65

Expected result

  • All events defined inside the array in monitorEvents() should be monitored

All events

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Focus the Command Line
  4. Type monitorEvents(document.getElementById("monitoredElement")) and hit Enter
  5. Click the blue <div> (#monitoredElement) above
    ⇒ The console should list a mouseover, several mousemove, a mousedown, a focus, a mouseup and a click event:
    mouseover clientX=<x coordinate>, clientY=<y coordinate> mousemove clientX=<x coordinate>, clientY=<y coordinate> ... mousedown clientX=<x coordinate>, clientY=<y coordinate> focus mouseup clientX=<x coordinate>, clientY=<y coordinate> click clientX=<x coordinate>, clientY=<y coordinate>
  6. Press A on the keyboard
    ⇒ The console should list a keydown, a keypress and a keyup event:
    keydown charCode=0, keyCode=65 keypress charCode=97, keyCode=0 keyup charCode=0, keyCode=65

Expected result

  • All events triggered by #monitoredElement should be monitored
