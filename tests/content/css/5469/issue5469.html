Issue 5469: Add units when changing the length '0' with up/down keys

Steps to reproduce

  1. Inspect the blue <div> above
  2. Switch to the Style side panel
  3. Click on the value of the width property to start inline editing
  4. Verify the results of pressing Up Arrow with the cursor at the 0 of various values:
    • "-1.1em 0" ⇒ "-1.1em 1em"
    • "0" ⇒ "1px"
    • "'a 0 b'" ⇒ "'a 1 b'"
    • "-moz-calc(11px * 0)" ⇒ "-moz-calc(11px * 1)"

Expected Results

  • When '0' is known to be a length, a suitable unit should be added when increasing it with the Up Arrow key.
