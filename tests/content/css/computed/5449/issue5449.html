Issue 5449: Show infotips in Computed side panel

Text

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Computed side panel
  3. Inspect the blue <div> with the Firebug icon above (#element)
  4. Hover the first font inside of the font-family property
    ⇒ An infotip with a sample of the font should be shown
  5. Hover the value of the background-color property (rgb(140,​ 140,​ 255))
    ⇒ An infotip with a sample of the color should be shown
  6. Click the twisty at the background-image property
    ⇒ The property tree should get expanded and show two style definitions: #element -moz-linear-gradient(135deg,​ #788CFF,​ #B4C8FF) issue5449.html (line 20) #element url("firebug.png") issue5449.html (line 8)
  7. Hover the -moz-linear-gradient part of the first #element style
    ⇒ An infotip with a preview of the color gradient should be shown
  8. Hover the url() of the second #element style
    ⇒ An infotip with a preview of the image should be shown

Expected Result

  • Infotips should be shown for fonts, colors, images and color gradients.
