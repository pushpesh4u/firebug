Issue 3652: CSS autocomplete is broken

Steps to reproduce

  1. Open Firebug and select the HTML panel
  2. Inspect the blue <div> above
  3. Select the Style side panel
  4. Click on the value green of the CSS property color to open the inline editor
  5. Press Y
    ⇒ YELLOW should be offered as value
  6. Press Up
    ⇒ YELLOWGREEN should be offered as value
  7. Press Down
    ⇒ YELLOW should be offered as value
  8. Press Home
  9. Press Up
    ⇒ whitesmoke (in lowercase) should be offered as value
  10. Press Down
    ⇒ yellow (in lowercase) should be offered as value

Expected Result

  • There should always be suggestions when pressing Up/Down
  • The suggestions should always represent the next value in alphabetical order
