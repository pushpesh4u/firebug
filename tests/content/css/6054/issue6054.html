Issue 6054: Colors are always copied in rgb() format

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Change the color display to Colors As Hex
  4. Inspect the blue <div> (#element) above
  5. Right-click the #element rule and choose Copy Declaration
  6. Paste the copied string to a text editor

Expected result

  • #element { width: 100px; height: 100px; background-image: -moz-linear-gradient(135deg, #788CFF, #B4C8FF); }
