Issue 6116: undefined values are ignored in string formatting of console.log

Steps to reproduce

  1. Open Firebug, select the Console panel and enable it
  2. Execute the following expression on the command line:
    console.log('%o%o%o', 'foo', undefined, 'bar')
  3. You should see following in the Console panel:
    "foo"undefined"bar"
