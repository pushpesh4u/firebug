Issue 6439: Consider precision for Console API %f log pattern

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Type console.log("amount: %f", 4.3152) into the Command Line and hit Enter
  4. Repeat step 3 for %.2f, %.1f, %.0f and %.5f

Expected result

  • The console output should look like this: amount: 4.3152 amount: 4.32 amount: 4.3 amount: 4 amount: 4.31520
