Issue 5382: Console.log display bug with nested arrays

Steps to reproduce

  1. Open Firebug, select the Console panel and enable it
  2. Execute the following expression on the command line:
    console.log( [['123'], '123'.match(/\w/)] )
    The second item in the array should be expandable. Clickin on the expand button (+) should show three entries)
  3. Execute the following expression on the command line:
    console.log( [1, window] )
    Clicking on the second item in the array "Window issue5382.html" should navigate to the DOM panel.
