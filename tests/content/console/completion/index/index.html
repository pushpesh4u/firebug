Index Completions

This test-case is intended for testing completions involving obj["member"] syntax, introduced in issues 3622 and 4803.
Simon Lindholm, simon.lindholm10@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Step by step type the following expressions in the command line and verify that completions appear for:
    • qqq
    • qqa
    • window.qqa
    • window['
    • window['qqq
    • window["qqq
    • window["qqa'
    • window['qqa\'
    and don't appear for:
    • -
    • "
    • \"
    • ["
    • qqq-
    • window.qqa'
    • window[
    • window['qqa'
    • window[/qq
  3. Type qqq into the command line, press TAB, and verify that it contains window["qqq--"].
  4. Type window.qqq into the command line, press TAB, and verify that it contains window["qqq--"].
  5. Type qqa into the command line, press TAB, and verify that it contains window["qqa'\"\\"].
  6. Type window['qqa into the command line, press TAB, and verify that it contains window['qqa\'"\\'].
  7. Type window["qqa into the command line, press TAB, and verify that it contains window["qqa'\"\\"].
