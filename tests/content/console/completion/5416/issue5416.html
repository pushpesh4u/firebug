Issue 5416: Auto-completion for Command Line API

Steps to reproduce

  • Open Firebug and enable the Console panel.
  • Verify the following completions by typing the prefix and pressing TAB:
    • "dirx" ⇒ "dirxml"
    • "if (false)dirx" ⇒ "if (false)dirxml"
    • "dirxml.to" ⇒ "dirxml.toString"
    • "copy" ⇒ "copy"
    • "window.copy" ⇒ "window.copy"
    • "window.dirx" ⇒ (no completion)
    • "traceA" ⇒ (no completion)
  • Verify that "cop" has a completion box consistent of "copx" and "copy", both regular properties, and that "und" has one consistent of "undebug" and "undefined" (the former an API completion, with italic style).

Expected Results

  • Members of the Command Line API should show up in completions, in italic font style
