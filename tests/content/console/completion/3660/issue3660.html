Issue 3660: Implement pageUp/pageDown and mouseWheel for completion list

Steps to reproduce

  • See issue3660.js.

Expected Results

  • Up/Down should move the selection one step up/down, and wrap at edges.
  • Page Up/Page Down should move the selection to the point in the completion popup where it is 2 steps away from the edge, or right to the edge if it is the start/end. If ahead of (as can happen with mousedown) or at that point, it should move to the next one. It should never wrap at edges, though.
  • Home/End should move the selection to the start or end of the completion list.
  • Scrolling with the mouse wheel should move the selection with standard number lines. It should not wrap at edges.
  • Mousing down on a completion should select it, and a full click should complete.
  • When the selection is set from anything but mousedown, the popup view area should adjust with as little as possible to show the selection and a padding of 2 completions around it.
