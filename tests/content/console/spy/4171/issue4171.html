Issue 4171: Expanding an XMLHttpRequest in Console tab before it ends makes JSON tab disappear

Steps to reproduce

  1. Open Firebug, switch to the Console panel and enable it.
  2. Click this button
  3. It generates a new XHR entry in the Console panel
  4. Expand the entry before the request finishes (you have about 5 sec)
  5. Only Headers info tab is displayed since the request didn't finish yet
  6. As soon as the response is received, both Response and JSON tabs should appear.
