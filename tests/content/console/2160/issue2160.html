This test-case is intended for Issue #2160 - Console does not always start auto-scrolled to the bottom.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug, enable and select the Console panel.
  2. Reload the test page. You should see bunch of logs in the panel.
  3. Scroll the Console panel to the top.
  4. Reload: the content must be again scrolled at the top.
  5. Scroll the Console panel to the bottom.
  6. Reload: the content must be again scrolled at the bottom.
