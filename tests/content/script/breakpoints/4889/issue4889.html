Issue 4889: Folding for breakpoint categories

Steps to reproduce

  1. Open Firebug
  2. Enable the Console and the Script panel
  3. Reload the page
  4. Switch to the Console panel
  5. Click the two buttons Call first non-existing function and Call second non-existing function
    ⇒ There should be two errors shown inside the console.
  6. Click the breakpoints besides the two errors
  7. Switch to the Script panel and there to the Breakpoints side panel
    ⇒ There should be an Error Breakpoints section containing two error breakpoints.
  8. Click the Error Breakpoints header
    ⇒ The section should be collapsed.
  9. Click the Error Breakpoints header again
    ⇒ The section should be expanded.

Expected result

  • Folding of breakpoint categories should work as expected.
