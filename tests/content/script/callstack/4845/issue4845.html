Issue 4845: Rerun should have a shortcut like the other debugging options

Steps to reproduce

  1. Open Firebug
  2. Enable the Console and the Script panel and switch to the Script panel
  3. Reload the page
    ⇒ The script execution should stop at line 11 (debugger;).
  4. Switch to the Console panel
    ⇒ The console should list a log message Hello Firebug user!.
  5. Switch back to the Script panel
  6. Press Shift+F8 to rerun the stack trace
  7. Switch to the Console panel again

Expected result

  • The console should list two logs of Hello Firebug user!, which indicates that the stack trace was executed again.
