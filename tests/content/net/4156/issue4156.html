Issue 4156: Multiple filters selectable in Net panel

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Net panel
  3. Reload the page
  4. Click the CSS filter button inside the Net panel, hold down Ctrl and click the JavaScript and Images filter button
    ⇒ All three buttons should be pressed and the Net panel should list three requests, one to testcase.css, one to issue4156-test.js and one to firebug.png.
  5. Hold down Ctrl and click the JavaScript filter button
    ⇒ The CSS and Images button should still be pressed and the Net panel should list two requests, one to testcase.css and one to firebug.png.
  6. Hold down Ctrl and click the CSS and the Images filter button
    ⇒ The All button should be pressed now and the Net panel should list all four requests, one to issue4156.html, one to testcase.css, one to issue4156-test.js and one to firebug.png.

Expected result

  • Multiple request filters should be activatable by pressing Ctrl while clicking the filter buttons.
