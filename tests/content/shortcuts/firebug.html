Test Case for Firebug shortcuts

This test-case is intended for Firebug shortcuts: F12 .
Jan Odvarko, odvarko@gmail.com

  1. Launch Firefox and make sure Firebug is deactivated for the current page.
  2. Pres F12, Firebug should be activated and the UI opened.
  3. Pres F12 again, Firebug UI should be minimized and Firebug still activated (Firebug icon colorful).
  4. Pres F12 again, Firebug UI should be opened, Firebug still activated.
  5. Pres Shift+F12, Firebug UI should be closed, Firebug deactivated.
