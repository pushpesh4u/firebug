Text Size

This test-case is intended for options: Text Size Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable/select the Console panel.
  2. Click on the Execute Test button below.
  3. You should see one log in the Console panel.
  4. Pick Increase Text Size, Decrease Text Size and Normal Text Size from Firebug -> Text Size menu.
  5. Verify that the text in the console panel properly changes its size.
