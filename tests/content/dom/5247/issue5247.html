Issue 5247: Add reps for different types of CSSRules

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Select issue5247.html from the Location Menu
  4. Right-click the Location Button and choose Inspect in DOM panel from the context menu
    ⇒ Firebug should switch to the DOM panel and display the stylesheet
  5. Expand the cssRules property
    ⇒ There should be 6 rules listed:
    CSSCharsetRule utf-8 CSSImportRule externalStylesheet.css CSSMediaRule (min-width: 500px) and (max-width: 700px) CSSFontFaceRule "TitilliumMaps" CSSKeyframesRule slidein CSSStyleRule #internalRule
  6. Expand the CSSKeyframesRule and the cssRules property in it
    ⇒ There should be 2 CSSKeyframeRules listed:
    CSSKeyframeRule 0% CSSKeyframeRule 100%

Expected result

  • Each CSSRule should have its own display.
